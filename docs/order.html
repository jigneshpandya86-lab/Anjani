📸 How to Update Promo Photos

  1. Upload photo to imgbb.com → copy Direct Link.
  2. Open script.js → find ANJANI_CONFIG.promos at the very top.
  3. Paste your URL into the image: '' field of the slide you want to update.
  4. Also update title, tag, desc text in the same block.
  5. Save & push to GitHub — live in ~2 minutes ✅
